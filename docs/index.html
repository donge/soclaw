🦞 安全运营龙虾

基于 OpenClaw 的智能安全运营助手，自动化风险分析、弱点评估、API业务识别

了解更多 GitHub

核心功能

🔍

风险事件研判

自动查询待处理风险事件，溯源分析访问记录和HTTP报文，智能判断风险真实性

⚠️

弱点事件分析

分析弱点触发时的HTTP流量，自动识别误报，确认真实安全问题

🔗

API业务识别

自动分析API的业务含义、参数、重要性等级，辅助配置防护策略

📱

应用系统识别

根据API列表自动识别应用名称和业务描述，管理应用配置

💬

多渠道接入

支持 Telegram、Discord、Slack、DingTalk、QQ 等多种聊天渠道

🔌

多LLM支持

支持 OpenAI、Anthropic、vLLM、Groq、Zhipu 等多种大模型

手动模式 (Manual)

  • ✓ LLM 分析后生成提案
  • ✓ 等待用户审批
  • ✓ 用户确认/忽略/修改参数
  • ✓ 执行相应操作

自动模式 (Auto)

  • ✓ LLM 直接分析
  • ✓ 自动确认/忽略
  • ✓ 无需人工干预
  • ✓ 适合低风险操作

Debug UI

浏览器可视化操作界面

通过 Web 界面直接与 Agent 对话，查看工具、技能，审批安全运营提案

  • ✓ 💬 对话 - 与 Agent 实时交流
  • ✓ 🔧 工具 - 查看可用工具列表
  • ✓ ✨ 技能 - 查看已加载技能
  • ✓ 📋 提案 - 审批安全运营提案
  • ✓ ⚙️ 设置 - 查看系统信息
# 启动 Debug UI
docker run -p 18789:18789 openclaw

# 或使用 docker-compose
docker-compose up -d

# 访问
http://localhost:18789

Docker 部署

快速开始

# 1. 克隆项目
git clone https://github.com/sipeed/openclaw.git
cd openclaw

# 2. 复制配置
cp docker-compose.example.yml docker-compose.yml

# 3. 编辑配置
# 添加你的 API Key 等

# 4. 启动
docker-compose up -d

# 5. 访问
# Gateway: http://localhost:18790
# Debug UI: http://localhost:18789
